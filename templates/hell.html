Добро пожаловать на страницу пет-роекта "Full-stack web project"!

Этот проект - это увлекательное приключение в мире разработки веб-приложений. Я создал полнофункциональное веб-приложение, которое будет обладать многими возможностями и функциями. Я использую широкий спектр технологий для создания этого приложения, от фронтенд-фреймворков до баз данных.

Основная цель проекта - научиться создавать веб-приложения с использованием всех необходимых технологий и инструментов. Я постараюсь создать что-то действительно полезное и удобное для пользователей.

В проекте я использую следующие технологии и инструменты:

  • Bootstrap и Tallwind для создания фронтенд-части приложения;
  • Flask и Django для создания бэкенд-части приложения;
  • SQlAlchemy и PostgreSQL для хранения данных;
  • Git для управления версиями кода;
  • Docker для упрощения процесса развертывания.
